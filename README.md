# NER filter

A filter list for my fellow Hungarians to block all the state-sponsored propaganda.

Compatible with most ad blockers, including uBlock Origin and Adblock Plus.

## Usage (easy)

[Subscribe to filter list](abp:subscribe?location=https://nerfilter.hu/list.txt&title=NER%20filter%20-%20System%20of%20Nathyonal%20Co-crime%20propaganda)

### uBlock Origin

1. Right-click the link above
2. Click the **Subscribe to filter list** button under the **uBlock Origin** menu
3. Click the **Subscrible** button in the top right corner of the new window

### Adblock Plus

1. Click the link above
2. Click the **Yes, add this filter list** button in the new window

## Usage (manual)

Simply add the following URL as a custom filter list:

```txt
https://nerfilter.hu/list.txt
```
