# NER filter

Egy szűrőlista magyar honfitársaimnak, mely tiltja az összes kormánypropagandát.

Kompatibilis a legtöbb hirdetésblokkolóval, ide értve a uBlock Origin-t és az Adblock Plus-t.

## Használat (egyszerű)

[Feliratkozás a szűrőlistára](abp:subscribe?location=https://nerfilter.hu/list.txt&title=NER%20filter%20-%20Nemzethy%20Egy%C3%BCttb%C5%B1n%C3%B6z%C3%A9s%20Rendszere%20propaganda)

### uBlock Origin

1. Kattints a fentebbi linkre jobb gombbal
2. Az **uBlock Origin** menüpont alatt kattints a **Feliratkozás szűrőlistára** (**Subscribe to filter list**) gombra
3. Kattints a megnyíló ablak jobb felső sarkában lévő **Feliratkozás** (**Subscribe**) gombra

### Adblock Plus

1. Kattints a fentebbi linkre
2. Kattints a megnyíló ablakban az **Igen, végezze el e szűrőlista hozzáadást** (**Yes, add this filter list**) gombra

## Használat (manuális)

Add hozzá ezt a listát az általad használt hirdetésblokkolóhoz:

```txt
https://nerfilter.hu/list.txt
```

## Hozzájárulás

Ha találsz egy propaganda weboldalt, mely nincs rajta a listán, nyugodtan nyiss egy [issue](https://github.com/nerfilter/nerfilter/issues/new)-t vagy [pull request](https://github.com/nerfilter/nerfilter/pulls)-et.
